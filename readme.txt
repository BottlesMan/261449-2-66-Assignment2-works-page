    ใน Triangle v001,v002 นั้น ได้ใช้ Black-Box Testing เนื่องจากว่าเราไม่สามารถเห็น code การทำงานเบื้องหลังได้ จึงจำเป็นต้องใช้ Black-Box Testing โดยจะทดสอบโดยการให้เข้าเว็บไซด์ของ Triangle และใส่ input ตามที่กำหนดไป จากนั้นก็กดปุ่มเพื่อแสดงผลลัพธ์ของรูปแบบสามเหลี่ยม

    ใน Triangle จาก Activity 1 นั้น เราได้ใช้ White-Box Testing เนื่องจากว่าเรามี code based มาให้แล้ว ดังนั้นเราสามารถทำ Test Case จาก code ได้ โดยจะเน้นทดสอบเฉพาะผลลัพธ์ของรูปแบบสามเหลี่ยมเท่านั้น โดยจะให้ตัวแปรของด้านทั้ง 3 ด้าน
